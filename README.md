
## O que tem no projeto

- CRUD de clientes
- CRUD de dívidas
- Banco de dados SQLite
- Validação com DataAnnotations
- Validação de CPF
- Validação de e-mail
- Idade calculada pela data de nascimento
- Listagem de clientes com busca por nome
- Paginação de 10 em 10
- Ordenação por quem mais deve
- Soma total das dívidas abertas
- Regra de apenas uma dívida aberta por cliente
- Script SQL em `schema.sql`

## Como executar

```bash
dotnet restore
dotnet run
```
